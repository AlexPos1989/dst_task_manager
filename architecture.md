# Архитектура

## Обзор
Приложение представляет собой одностраничное React-приложение, отображающее сетку боссов из игры "Don't Starve Together" (DST). Пользователи могут кликнуть по боссу, чтобы отметить его как "убитого" (зачеркнутого). Состояние сохраняется в локальном хранилище браузера.

## Компоненты
- **App**: Основной контейнер, управляющий состоянием убитых боссов.
- **BossGrid**: Компонент сетки для отображения боссов.
- **BossCard**: Компонент, представляющий отдельного босса, отображающий его изображение и имя, а также обрабатывающий взаимодействие по клику.
- **GuideModal**: Модальное окно, отображающее подробную информацию о боссе, включая руководство по призыву и список наград.

## Данные
- **Список боссов**: Константный массив объектов боссов, каждый из которых содержит:
  - `id`: Уникальный идентификатор (строка).
  - `name`: Отображаемое имя (строка).
  - `imageUrl`: URL изображения босса (строка).
  - `wikiUrl`: URL вики-страницы босса (строка).
  - `description`: Краткое описание (строка).
  - `guide`: Руководство по призыву/бою (строка).
  - `loot`: Список выпадающих предметов (массив строк).
  - `group`: Категория босса (строка).

## Управление состоянием
- **Local Storage**: Используется для сохранения списка ID убитых боссов.
- **React State**: `killedBosses` (Set<string>) для отслеживания состояния текущей сессии.
- **WebSockets**: Используются для real-time синхронизации состояния между клиентами.
- **Группы боссов**: `surface`, `caves`, `lunar`, `hamlet`, `shipwrecked`, `event`, `sea`, `seasonal`, `miniboss`.

## Стилизация
- **Tailwind CSS**: Используется для всей стилизации.
- **Дизайн**:
  - Темная тема, соответствующая эстетике игры.
  - Сетка для адаптивности.
  - Визуальные подсказки для состояния "убит" (оттенки серого, зачеркивание, уменьшение непрозрачности).

## Технологический стек
- React 18+
- TypeScript
- Vite
- Tailwind CSS
